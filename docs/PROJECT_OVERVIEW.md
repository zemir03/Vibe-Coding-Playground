# Vibe Coding Playground — Project Overview

## Что это

**Vibe Coding Playground** — одностраничный showcase-сайт, демонстрирующий возможности современной веб-разработки с акцентом на анимации, интерактив и визуальные эффекты. Создан как витрина вайбкодинга — подхода, где ИИ-инструменты (Claude, Cursor) позволяют создавать впечатляющие продукты за часы, а не недели.

## Цели

| Приоритет | Цель |
|-----------|------|
| 🔴 MVP | Showcase 5 ключевых возможностей веб-разработки в отдельных секциях |
| 🟡 Must | Тёмная/светлая тема, полная адаптивность, плавные переходы |
| 🟢 Nice | Кастомный курсор, Easter eggs, шаринг в соцсети |

## Целевая аудитория

**Первичная:** Ученики курса по вайбкодингу
- Хотят понять что возможно создать
- Ищут вдохновение и доказательство концепции
- Уровень: новичок → middle

**Вторичная:** Опытные разработчики
- Оценивают качество ИИ-генерации
- Ищут идеи для своих проектов
- Могут форкнуть репо

## Ключевые метрики успеха

- [ ] Первое впечатление: "Вау, это сделал ИИ?" 
- [ ] Время до первого взаимодействия < 3 сек
- [ ] Lighthouse Performance ≥ 85
- [ ] Работает на мобилках (iOS Safari, Android Chrome)
- [ ] Анимации не лагают (60fps)

## Секции сайта

| # | Секция | Основная фича |
|---|--------|---------------|
| 1 | **Hero** | Анимированный заголовок + particle-фон |
| 2 | **Animations** | Stagger, scroll-trigger, hover-reactions |
| 3 | **Interactivity** | Конфетти, морфинг кнопок, слайдеры |
| 4 | **Visual Effects** | Glassmorphism, glow, gradient magic |
| 5 | **Responsive** | Превью на разных устройствах |
| 6 | **Footer** | CTA + GitHub ссылка |

## Tone & Feel

> "Не корпоративный сайт. Не portfolio. Это — ОПЫТ. Каждая секция должна вызывать желание нажать что-нибудь."

- Энергичный, но не кричащий
- Тёмная тема как основа (ощущение ночного клуба для разработчиков)
- Неоновые акценты: cyan + violet + electric green
- Движение везде, но оправданное — не ради движения
